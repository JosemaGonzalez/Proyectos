Animaciones

Una animación es un efecto dinámico que cambia las propiedades de un elemento en el tiempo y que puede tener varios estados intermedios especificados

Concepto previo: fotograma clave

Los fotogramas claves son valores iniciales y finales que debe tener la animación CSS.

                    @keyframes anim1 {
                        0% {
                              ...
                        }
                        50% {
                              ...
                        }
                        100% {
                              ...
                        }
                    }

animation-name

Nombre del fotograma clave.

                    #animacion1 {
                        animation-name: anim1;
                        ...
                    }
                
                    @keyframes anim1 {
                        from {
                            left: 0px;
                        }
                        to {
                            left: 100px;
                        }
                    }
Nombre de la animación: anim1

animation-duration

Duración de la animación

                    #caja_animada{
                        animation-name: anim2;
                        animation-duration:2s;
                        ...
                    }

Animación de 2 segundos de duración

animation-iteration-count

Veces que se repite la animación. Se puede poner:

- número: número de veces que se va a repetir la animación

- infinite: la animación se repetirá infinitas veces

- initial: establece esta propiedad a su valor predeterminado

- inherit: hereda esta propiedad del padre

                    #caja_animada2{
                        animation-name: anim2;
                        animation-duration:2s;
                        animation-iteration-count: infinite;
                        ...
                    }
Animación que se repite 5 veces
Animación que se repite infinitas veces

animation-timing-function

Especifica la velocidad en que se desarrollará la animación. Puede ser:

- ease: este es el valor por defecto. Tiene un comienzo lento, luego rápido y termina de nuevo lentamente.

- linear: la misma velocidad durante toda la duración de la transición.

- ease-in: efecto de transición con comienzo lento.

- ease-out: efecto de transición con comienzo rápido y final lento.

- ease-in-out: efecto de transición con comienzo y final lento; más rápido en medio.

- cubic-bezier(n,n,n,n): define tus propios valores para la curva de Bezier. Cada valor es entre 0 y 1.

- initial: establece esta propiedad a su valor por defecto.

- inherit: hereda esta propiedad del elemento padre.

                    #caja_animada3{
                        animation-name: anim3;
                        animation-duration:5s;
                        animation-iteration-count: infinite;
                        animation-timing-function: linear;
                        ...
                    }

animation-direction

Esta propiedad define el sentido de la animación. Puede ser:

- normal: cada vez que termina un ciclo, la animación se reinicia al estado inicial y comienza desde el principio. Este es el comportamiento por defecto.

- alternate: la animación, al terminar un ciclo, invierte su dirección. Es decir, los pasos de la animación se ejecutan al revés.

- reverse: cada ciclo de la animación se reproduce al revés . Cada vez que comienza un ciclo de animación, ésta se posiciona en el estado final y comienza desde ahí.

- alternate-reverse: es similar a alternate pero la animación se reproduce al revés. Es decir la animación se posiciona en el estado final, comienza a reproducirse al reves y, cuando llega al inicio vuelve a reproducirse de forma normal hasta llegar al final de la secuencia. Y vuelve otra vez a repetirse.

                    #caja_animada4{
                        animation-name: anim4;
                        animation-duration:5s;
                        animation-iteration-count: infinite;
                        animation-direction: normal;
                        ...
                    }

animation-delay

Propiedad que nos indica el momento en el que comenzará la animación. Si el valor es 0 se ejecuta en cuanto se carga la página.

                    #caja_animada5{
                        animation-name: anim5;
                        animation-duration:5s;
                        animation-delay:2s;
                        ...
                    }

  1. onclick, Pulsar el ratón
  2. ondblclick, Pulsar dos veces con el ratón
  3. onkeydown, Pulsar una tecla y no soltarla
  4. onkeypress, Pulsar una tecla
  5. onkeyup, Soltar una tecla pulsada

  7. onload, Página cargada completamente
  8. onunload, Se abandona la página

  11. onmouseup, Soltar el botón del ratón

  13. onmouseout, El ratón sale del elemento
  14. onmouseover, El ratón entra en el elemento
  15. onblur, Deseleccionar el elemento
  16. onfocus, Seleccionar un elemento
  17. onchange, Deseleccionar un elemento que se ha modificado
  18. onresize, Modificar el tamaño de la ventana
  19. onselect, Seleccionar un texto
